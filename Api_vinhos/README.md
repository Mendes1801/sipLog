# 🍷 API de Vinhos (Módulo de Integração)

Módulo ou script em Java dedicado à conexão, consumo e parseamento de informações de uma base de dados externa de vinhos.

## Utilidade
Permite consultar informações padronizadas sobre garrafas, uvas, teor alcoólico e origens, evitando que os usuários do SipLog precisem preencher manualmente todos os dados de uma bebida comercial existente.

## Compilação
Sendo um projeto Maven padrão (contém `pom.xml`):
```bash
mvn clean package
```
